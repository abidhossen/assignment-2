PROJECT NAME:
Intrinsic Vehicle Renting Service

Live URL: https://assignment-2-delta-five.vercel.app/api/v1

This is a vehicle rent managing service. Here user can register and rent their desired vehicle. Admin will manage the whole the system. Admin can add vehicles, delete vehicles, update vehicles. He can view all the users and update or delete any user. User can update their profile or cancel any booking.

Technology used in the project:
Nodejs, Express.js, Typescript, BcryptJs, NeonDB, SQL, Vercel

Setup: Modular pattern is used in the whole project.

Main app is running in the app.ts
Our server is running through the server.ts
Under Src folder we have modules, helpers, config, middleware
